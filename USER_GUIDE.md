## Premiers pas

### Création de compte

Au premier lancement de l'application, vous devez créer un compte.

1. **Cliquez sur "S'inscrire"** sur l'écran de connexion
2. **Sélectionnez votre type de compte** :
   - **Patient** : Pour suivre vos symptômes et votre santé
   - **Professionnel** : Pour accompagner des patients

3. **Remplissez le formulaire** :
   - Prénom et nom
   - Email (utilisé pour la connexion)
   - Téléphone
   - Date de naissance
   - Mot de passe (minimum 6 caractères)
   - Spécialisation (pour les professionnels uniquement)

4. **Validez** en cliquant sur "S'inscrire"

### Connexion

1. Entrez votre **email** et **mot de passe**
2. Cliquez sur **"Se connecter"**
3. Vous serez redirigé vers votre tableau de bord

## Interface Patient

### Tableau de bord

Le tableau de bord patient affiche :
- Un message de bienvenue personnalisé
- Des raccourcis vers les fonctionnalités principales
- Un accès rapide aux actions courantes

#### Raccourcis disponibles
- **Ajouter un symptôme** : Enregistrer un nouveau symptôme
- **Mes exercices** : Consulter les exercices recommandés
- **Rendez-vous** : Voir votre calendrier médical
- **Mes rapports** : Accéder à vos rapports de progrès
- **Forum** : Participer aux discussions communautaires
- **Messages** : Communiquer avec votre professionnel de santé

### Suivi des symptômes

#### Ajouter un symptôme

1. Accédez à **"Symptômes"** depuis le menu principal
2. Cliquez sur le bouton **"+"** en bas à droite
3. Remplissez le formulaire :
   - **Type de symptôme** : Fatigue, Douleur, Troubles visuels, etc.
   - **Sévérité** : Échelle de 1 à 10
   - **Description** : Détails sur le symptôme
   - **Date et heure** : Quand le symptôme est apparu
   - **Durée** : Combien de temps il a duré (optionnel)
   - **Déclencheurs** : Facteurs ayant pu provoquer le symptôme

4. Cliquez sur **"Enregistrer"**

#### Consulter l'historique

- La liste affiche tous vos symptômes enregistrés
- Utilisez le **filtre** (icône en haut) pour trier par type
- **Tirez vers le bas** pour actualiser la liste
- **Cliquez sur un symptôme** pour voir les détails
- **Supprimez** un symptôme en cliquant sur l'icône de corbeille

### Exercices

#### Consulter les exercices

1. Accédez à **"Exercices"** depuis le menu
2. Parcourez la liste des exercices disponibles
3. Utilisez les **filtres** pour trier par :
   - Catégorie (Mobilité, Équilibre, Force, etc.)
   - Difficulté (Facile, Moyen, Difficile)

#### Détails d'un exercice

Cliquez sur un exercice pour voir :
- **Description complète**
- **Instructions étape par étape**
- **Durée estimée**
- **Bénéfices attendus**
- **Précautions à prendre**
- **Image ou vidéo** (si disponible)

### Calendrier et rendez-vous

#### Consulter le calendrier

1. Accédez à **"Calendrier"** depuis le menu
2. Le calendrier affiche :
   - Vue mensuelle par défaut
   - Points sur les jours avec rendez-vous
   - Jour actuel mis en évidence

#### Voir les rendez-vous d'un jour

1. **Cliquez sur une date** dans le calendrier
2. Les rendez-vous du jour s'affichent en dessous
3. Chaque rendez-vous montre :
   - Titre et description
   - Horaire (début - fin)
   - Statut (Planifié, Confirmé, Complété, Annulé)
   - Lieu (si renseigné)

### Messages et chat

#### Démarrer une conversation

1. Accédez à **"Messages"** depuis le menu
2. Cliquez sur le bouton **"+"** en bas à droite
3. Sélectionnez un **professionnel de santé** dans la liste
4. La conversation s'ouvre automatiquement

#### Envoyer un message

1. Tapez votre message dans le champ en bas
2. Appuyez sur **Entrée** ou cliquez sur l'icône **"Envoyer"**
3. Le message apparaît immédiatement dans la conversation

#### Recevoir des messages

- Les nouveaux messages apparaissent **en temps réel**
- Un **badge rouge** indique le nombre de messages non lus
- Les conversations sont triées par date du dernier message

### Forum communautaire

#### Consulter les publications

1. Accédez au **"Forum"** depuis le menu
2. Parcourez les publications des autres membres
3. Chaque publication affiche :
   - Nom de l'auteur
   - Date de publication
   - Catégorie
   - Contenu
   - Nombre de likes

#### Créer une publication

1. Cliquez sur le bouton **"+"** en bas à droite
2. Remplissez :
   - **Titre** : Sujet de votre publication
   - **Contenu** : Votre message
   - **Catégorie** : Général, Symptômes, Traitements, etc.
3. Cliquez sur **"Publier"**

### Rapports de progrès

#### Générer un rapport

1. Accédez à **"Rapports"** depuis le menu
2. Cliquez sur l'icône **"+"** en haut à droite
3. Choisissez le type de rapport :
   - **Hebdomadaire** : 7 derniers jours
   - **Mensuel** : 30 derniers jours
4. Le rapport est généré automatiquement

#### Consulter un rapport

Chaque rapport contient :
- **Période couverte**
- **Nombre de symptômes** enregistrés
- **Exercices complétés**
- **Sévérité moyenne** des symptômes
- **Symptôme le plus fréquent**
- **Pourcentage de progrès**

### Profil

#### Consulter votre profil

1. Cliquez sur l'icône **"Profil"** (personne) en haut à droite
2. Vous verrez :
   - Votre avatar (initiales)
   - Nom complet
   - Rôle
   - Email, téléphone, date de naissance
   - Date de création du compte

#### Modifier votre profil

- **Modifier le profil** : Changer vos informations (à venir)
- **Changer le mot de passe** : Sécuriser votre compte (à venir)

#### Déconnexion

1. Cliquez sur **"Déconnexion"**
2. Confirmez votre choix
3. Vous serez redirigé vers l'écran de connexion

Le tableau de bord professionnel affiche :
- Message de bienvenue avec votre spécialisation

#### Raccourcis disponibles
- **Créer un exercice** : Ajouter un nouvel exercice

### Gestion des exercices

#### Créer un exercice

1. Accédez à **"Exercices"** depuis le menu
2. Cliquez sur le bouton **"+"**
3. Remplissez le formulaire :
   - **Titre** : Nom de l'exercice
   - **Description** : Objectif et présentation
   - **Catégorie** : Type d'exercice
   - **Difficulté** : Niveau requis
   - **Durée** : Temps estimé en minutes
   - **Instructions** : Étapes détaillées
   - **Bénéfices** : Avantages attendus
   - **Précautions** : Avertissements
   - **Image/Vidéo** : Support visuel (optionnel)

### Communication avec les patients

#### Consulter les conversations

1. Accédez à **"Messages"** depuis le menu
2. Vous voyez la liste de vos conversations avec les patients
3. Les messages non lus sont en **gras** avec un badge

#### Répondre à un patient

1. Cliquez sur une conversation
2. Tapez votre réponse
3. Envoyez le message

#### Initier une conversation

1. Cliquez sur le bouton **"+"**
2. Sélectionnez un patient dans la liste
3. Commencez à échanger

### Gestion du calendrier

Les professionnels peuvent :
- Consulter leur planning de rendez-vous
- Voir les détails de chaque consultation
- Suivre le statut des rendez-vous

## Questions fréquentes

### Comment récupérer mon mot de passe ?

Actuellement, la fonctionnalité de récupération de mot de passe n'est pas implémentée. Contactez un administrateur pour réinitialiser votre compte.

### Puis-je utiliser l'application hors ligne ?

Oui ! L'application fonctionne entièrement hors ligne. Toutes vos données sont stockées localement sur votre appareil.

### Mes données sont-elles synchronisées entre appareils ?

Non, actuellement les données sont stockées uniquement sur l'appareil local. La synchronisation cloud est prévue dans une future version.

### Comment supprimer mon compte ?

Contactez un administrateur pour supprimer votre compte et vos données.

### Les messages sont-ils vraiment en temps réel ?

Oui, le système de chat vérifie les nouveaux messages toutes les 2 secondes pour simuler le temps réel.

### Puis-je exporter mes rapports ?

La fonctionnalité d'export PDF est prévue dans une prochaine version.

### Comment signaler un bug ?

Contactez l'équipe de développement avec une description détaillée du problème rencontré.

### L'application est-elle disponible en plusieurs langues ?

Actuellement, l'application est disponible uniquement en français. Le support multilingue est prévu.

## Conseils d'utilisation

### Pour les patients

1. **Enregistrez vos symptômes régulièrement** pour un suivi précis
2. **Notez les déclencheurs** pour identifier les patterns
3. **Consultez vos rapports** pour suivre votre évolution
4. **Communiquez avec votre professionnel** en cas de questions
5. **Participez au forum** pour échanger avec d'autres patients

### Pour les professionnels

1. **Créez des exercices personnalisés** adaptés à chaque patient
2. **Consultez régulièrement les rapports** de vos patients
3. **Répondez rapidement** aux messages des patients
4. **Planifiez les rendez-vous** à l'avance
